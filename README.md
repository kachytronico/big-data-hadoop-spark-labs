## English summary

This repository is a curated portfolio of Big Data notebooks developed in Google Colab as part of an AI & Big Data specialization. It covers ETL, Hadoop, HDFS, MapReduce, Pig, Hive, and Spark through applied practice labs. Some notebooks use a Type 1 Diabetes dataset as an educational case study — for learning purposes only, not medical advice. The goal is to document practical data engineering skills in a clean and honest way, without overstating results or publishing raw course assignments.
